# Plan 10-03: Remove size Attribute from NuevoTramiteModal Select

## Objective

Make the client select dropdown in NuevoTramiteModal close immediately after selection by removing the `size` attribute that forces it to render as a permanent listbox.

## Context

The `<select>` element for "Buscar cliente existente" in `NuevoTramiteModal.tsx` has `size={Math.min(5, clientesFiltrados.length + 1)}`, which renders it as an always-open listbox instead of a native dropdown. When the user selects a client, the listbox stays visible. Removing the `size` attribute restores native dropdown behavior where the list closes on selection. The `onChange` handler and `clienteId` state already work correctly -- the only issue is the persistent open state caused by `size`.

## Tasks
